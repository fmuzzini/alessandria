Alessandria - Biblioteca Ebook

Questo è un programma per la gestione, la catalogazione e la visualizzazione di ebook.

Struttura del programma

Il progrmma è stato pensato come due aree distinte:
una è rappresentata dall'oggetto Biblioteca che immagazzina i libri,
l'altra è rappresentata dall'oggetto TabellaBiblioteca che fornisce una rappresentazione grafica dei contenuti della Biblioteca.

Ogni volta che nella Biblioteca avviene un cambiamento viene informata di ciò la TabellaBiblioteca che si comporterà di conseguenza

Pulsanti Laterali

I pulsanti laterali consentono all'utente di interagire con la biblioteca.
Ogni volta che la selezione della TabellaBiblioteca cambia questi pulsanti reagiscono:
quando è selezionato un solo libri è possibile modificarlo, eliminarlo e visualizzarlo
quando sono selezionati più libri è possibile solo eliminarli
mentre l'operazione di aggiunta di un libro è sempre possibile

Aggiunta del libro

Si può scegliere se selezionare manualmente i file o inserire intere cartelle
nel primo caso l'utente seleziona uno o più file da aggiungere
mentre nel secondo caso l'utente seleziona che tipi di file includere (tramite estensione), se includere le sottocartelle e la cartella da cui partire. Il programma seleziona i file con il formato specificato automaticamente. Dopo la selezione dei file, per ogni file, viene chiesto di aggiungere le informazioni opportune

Salvataggio e caricamento biblioteca

Il programma si avvia inizializzando una biblioteca vuota.
L'utente ha la possibilità di caricare una biblioteca già esistente dal menù file.
Una volta efettuate inserimenti e modifiche l'utente ha la possibilità di salvare la biblioteca su file sempre attraverso il menù file.
I file di Biblioteca Alessandria sono contraddistinti da estensione .ba
